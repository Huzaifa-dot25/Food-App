# Food-App
A full-stack Food Delivery Mobile Application built with React Native, ASP.NET Core Web API, and SQL Server. Features include customer ordering, restaurant management, delivery tracking, secure authentication, real-time notifications, and role-based dashboards.

# 🍔 Food Delivery Mobile Application

A modern, scalable, and feature-rich **Food Delivery Mobile Application** that connects customers, restaurants, delivery riders, and administrators on a single platform. The application provides a seamless food ordering experience with secure authentication, real-time order tracking, role-based dashboards, and an intuitive mobile interface.

## 🚀 Features

### 👤 Customer

* User Registration & Login
* Restaurant & Food Search
* Categories & Filters
* Shopping Cart
* Secure Checkout
* Multiple Payment Options
* Live Order Tracking
* Order History
* Favorites
* Ratings & Reviews
* Push Notifications
* Profile Management

### 🍽️ Restaurant Owner

* Restaurant Management
* Menu & Category Management
* Food Management
* Order Management
* Sales Dashboard
* Customer Reviews
* Business Profile

### 🛵 Delivery Rider

* Rider Dashboard
* Accept/Reject Deliveries
* GPS Navigation
* Delivery Status Updates
* Delivery History
* Earnings Overview

### 🛠️ Administrator

* Dashboard & Analytics
* User Management
* Restaurant Management
* Rider Management
* Order Management
* Category & Menu Management
* Coupon & Promotion Management
* Payment Monitoring
* Reports & Analytics
* System Settings

## 📱 Project Goals

* Deliver a smooth and responsive mobile experience.
* Build a secure and scalable backend architecture.
* Support multiple user roles with dedicated dashboards.
* Provide real-time order tracking and notifications.
* Follow clean architecture and industry best practices.

This project demonstrates full-stack mobile application development, modern software architecture, secure API design, and scalable system implementation suitable for real-world food delivery platforms.
